為您的 WooCommerce 網站添加 黑貓Pay 金流支付方式。

== Description ==

CCat Payments Gateway 提供多種台灣本地支付方式：

* 信用卡支付
* 超商代碼付款（全家/7-11/萊爾富）
* ATM 轉帳
* 超商條碼支付
* 行動支付 (OPW)
* 電子支付 (iCash)

特色：

* 支援電子發票開立
* WooCommerce Blocks 整合支援
* 完整中文化介面

== Installation ==

1. 上傳 `woocommerce-gateway-ccat` 資料夾至 `/wp-content/plugins/` 目錄
2. 在後台啟用插件
3. 前往 WooCommerce > 設定 > 黑貓Pay 設定您的金流

== Frequently Asked Questions ==

= 支援哪些 WooCommerce 版本？ =
本插件支援 WooCommerce 3.0 以上版本。

= 如何設定電子發票？ =
前往 WooCommerce > CCat 設定 > 發票設定 啟用並設定。
